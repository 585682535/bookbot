# Интеграция с ЮKassa

Для подключения онлайн-оплаты в BookBot CRM выполните следующие шаги:

1. **Зарегистрируйтесь в ЮKassa:**
   Перейдите на [yookassa.ru](https://yookassa.ru/) и создайте аккаунт для вашего бизнеса.

2. **Получите Shop ID:**
   После регистрации в личном кабинете ЮKassa вы найдете ваш `Shop ID` в разделе "Настройки" -> "Магазин".

3. **Сгенерируйте Secret Key:**
   В разделе "Настройки" -> "Ключи API" создайте новый секретный ключ. Сохраните его, так как он будет показан только один раз.

4. **Настройте Webhook:**
   В личном кабинете ЮKassa перейдите в раздел "Настройки" -> "HTTP-уведомления".
   Укажите URL для уведомлений: `https://ваш-домен.app/api/webhooks/yookassa`.
   Выберите событие: `payment.succeeded`.

5. **Введите данные в CRM:**
   Перейдите в настройки вашего бизнеса в BookBot CRM, включите "Онлайн-оплату" и введите полученные `Shop ID` и `Secret Key`.

# Промокоды

Вы можете создавать промокоды в разделе "Промокоды" вашей CRM.
- Поддерживаются скидки в процентах (%) или фиксированной сумме (₽).
- Можно ограничить срок действия и количество использований.
- Можно привязать промокод к конкретным услугам.

# Отзывы

Клиенты могут оставлять отзывы после посещения.
- Все новые отзывы попадают на модерацию в раздел "Отзывы".
- Вы можете одобрять, отклонять или отвечать на отзывы.
- Одобренные отзывы автоматически отображаются на вашей странице записи.
